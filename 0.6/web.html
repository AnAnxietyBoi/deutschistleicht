Zasluge Sich Glagoli

Pomoč? Nasveti:
- če hočeš glagol vpisati, klikni VPIŠI GLAGOLSKO OBLIKO
- če hočeš imeti glagol podan, klikni POKAŽI GLAGOLSKO OBLIKO
- če nočeš imeti opravka s stolcem, klikni PRESKOČI
- za hiter preimik med vpisnimi okni klikni CTRL
- namesto gumba preveri/nadaljuj lahko uporabiš ENTER

Nemške črke:
ä = Alt + a
ö = Alt + o
ü = Alt + u
ß = Alt + s
